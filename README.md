# MyWebDomains
> Checks to see what web domains are associated with a DNS scan.

## Requirements
- git
- python3
- requirements.txt


## Install instructions
1. Create a venv.
```sh
python -m venv venv
source venv/bin/activate
```

2. Install requirements
```sh
pip install -r requirements.txt
```

3. Run the tool
```sh
python main.py domain.com
```
